A community-curated list of articles, talks, videos and tutorials that cover Triggery. The list is intentionally empty until the 1.0 release — Triggery is pre-1.0 and the API can still change between minor versions, so we'd rather not point readers at content that ages out in a release cycle.

<Aside type="tip" title="Add your link">
If you've written about Triggery, given a talk, or recorded a tutorial — open a PR adding it here. Anything that helps the next person learn the library is in scope. Use the [`awesome-template.md`](https://github.com/triggeryjs/triggery/tree/main/.github/ISSUE_TEMPLATE) issue form, or open a direct PR against this page.
</Aside>
