API Reference покрывает каждый пакет, опубликованный под скоупом `@triggery/*`. Для каждого символа отведена своя страница со следующим содержимым:

- Полная TypeScript-сигнатура (с Twoslash-превью типов при наведении).
- Таблицы параметров и типа возвращаемого значения.
- Идиоматические и пограничные примеры.
- Перекрёстные ссылки на связанные символы.
