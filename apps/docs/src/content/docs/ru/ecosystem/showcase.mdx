Triggery всё ещё pre-1.0, и публичная витрина пока пуста. Если ты отгрузил что-то с Triggery — пет-проект, фича внутри большого приложения, демо — будем рады показать это здесь.

<Aside type="tip" title="Отправить свой проект">
Открой PR через [`showcase-template.md`](https://github.com/triggeryjs/triggery/tree/main/.github/ISSUE_TEMPLATE) — название, однострочное описание, ссылка, опционально скриншот. Принимаем всё, что использует Triggery не тривиально (что-то посерьёзнее примера со счётчиком).
</Aside>
