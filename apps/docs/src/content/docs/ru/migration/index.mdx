Большинство тех, кто переходит на Triggery, приходит с багажом: `useEffect` повсюду, пара саг, несколько регистраций в `listenerMiddleware`. Эти страницы объясняют, как каждая исходная парадигма ложится на модель «событие → условия → действия», и где codemod-скрипт может взять на себя скучную работу.
