Triggery adopts the **Contributor Covenant** (version 2.1) as its code of conduct, without modification. Every space the project runs in — repositories, discussions, issues, pull requests, Discord (once open), and any organised events — is covered.

- **Canonical text**: [contributor-covenant.org/version/2/1/code_of_conduct/](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
- **Plain-text version**: [code_of_conduct.md](https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.md)
- **Translations**: [contributor-covenant.org/translations/](https://www.contributor-covenant.org/translations/)

## Reporting

Reports of unacceptable behaviour go to the project maintainer at **`a@skhom.ru`** with the subject line `[triggery conduct]`. All reports are reviewed and investigated promptly and fairly. The maintainer is obligated to respect the privacy and security of the reporter.

The full text of the project's adoption — including scope and enforcement details — lives in [`CODE_OF_CONDUCT.md`](https://github.com/triggeryjs/triggery/blob/main/CODE_OF_CONDUCT.md).
