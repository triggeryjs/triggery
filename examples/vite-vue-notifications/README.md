# Triggery — notifications (Vite + Vue)

Same trigger as the React and Solid notification examples, Vue 3 + `<script setup>` components.

## Try it

- **Open in StackBlitz** — <https://triggeryjs.github.io/play/vite-vue-notifications/>
- **Read the recipe** — <https://triggeryjs.github.io/recipes/vue/notification-pipeline/>

Or run it locally:

```bash
pnpm install
pnpm dev
```
