# Triggery — onboarding tour (Vite + React)

A two-step product tour driven entirely by one trigger. Each step component only reports user activity (`route:dashboard-entered`, `feature:search-used`). The trigger handler decides which coachmark to show based on persisted progress conditions.

## Try it

- **Open in StackBlitz** — <https://triggeryjs.github.io/play/vite-react-onboarding/>
- **Read the recipe** — <https://triggeryjs.github.io/recipes/react/onboarding-flow/>

Or run it locally:

```bash
pnpm install
pnpm dev
```
