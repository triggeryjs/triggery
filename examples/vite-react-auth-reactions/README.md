# Triggery — auth reactions (Vite + React)

Session expired or a role revoked — the trigger decides the appropriate response based on conditions registered by independent features (is there a dirty form? which screens are open?).

Saves drafts, closes dependent routes, redirects, shows a toast — all from one trigger file.

## Try it

- **Open in StackBlitz** — <https://triggeryjs.github.io/play/vite-react-auth-reactions/>
- **Read the recipe** — <https://triggeryjs.github.io/recipes/react/auth-reactions/>

Or run it locally:

```bash
pnpm install
pnpm dev
```
